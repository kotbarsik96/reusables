Иконки:
Хранятся в /assets/icons в виде .svg файлов
Для корректного использования, после экспорта каждой иконки нужно:

1. удалить атрибуты width и height у svg
2. назначить svg id="icon-<название иконки>"
2.1 (необязательно) поменять значение атрибута fill на currentColor
3. для использования: <svg><use xlink:href="/assets/icons/<название иконки>.svg#icon-<название иконки>"></use></svg>
